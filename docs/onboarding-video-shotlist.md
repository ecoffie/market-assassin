# Mindy Onboarding — Storyboard & Shot List

Record against this. Left = exactly what to click/show on screen. Right = the beat's
purpose + on-screen callout to add in edit. Narration lives in the separate
teleprompter file (`onboarding-video-teleprompter.md`) — read that; watch this.

**Setup before you hit record (all 3 videos):**
- One demo company across all three: **"Summit Mechanical" — commercial HVAC install/service for federal buildings.** Same UEI, same Vault, same story. Continuity sells it.
- Clean browser: no bookmarks bar, no extensions visible, 1280×720 or 1920×1080, zoom 100%.
- Log in as the demo account BEFORE recording so no login screen leaks.
- Cursor: move slowly and deliberately. Pause 1s on a target before clicking so the edit can add a callout.
- Record each beat as its own clip — easier to re-take one beat than the whole video.

## VIDEO 1 — Set up your Market Profile  (My Vault · 4–5 min)

| # | ON SCREEN — do this | Purpose / edit callout |
|---|---------------------|------------------------|
| 1 | **Hook shot:** land on Today's Intel with a THIN feed (few matches). Hold 2s. | Sets up the problem: "one code = you miss stuff." |
| 2 | Open **Onboarding / Profile** → the "describe what you do" text box. Click into it. | Beat 1 start. |
| 3 | **Type slowly:** *"we install and service commercial HVAC systems for federal buildings."* | Show it's plain English, no codes. |
| 4 | Hit submit → let the **scan animation** play fully (Crosswalking NAICS↔PSC, Loading forecasts, Indexing pain points). Don't cut it short. | The "watch it work" moment. Callout: **"1 word → the whole market"** |
| 5 | Reveal the **derived NAICS set** — scroll it so the count of codes is visible. | Proof it found more than one code. Callout: **"Obvious code = 28%. Mindy finds the other 72%."** |
| 6 | Scroll to **keyword chips.** Hover a good one ("HVAC controls"), then remove a too-broad one ("services"). | Beat 2. Show tuning, not just accepting. Callout: **"These chips = what your alerts search"** |
| 7 | Navigate to **My Vault** (sidebar → Account → My Vault). | Beat 3 start. |
| 8 | Click the **UEI field**, type/paste the demo UEI, show the registration pull-in. | Vault = Mindy's memory. |
| 9 | Show **certifications** (check 8(a) or SDVOSB on the demo). | Set-aside flagging. |
| 10 | Click **"add past performance"** → show ONE real-looking project filled in. Don't type the whole thing on camera — have it pre-filled and just scroll/point. | Powers bid/no-bid + drafts. Callout: **"This powers your bid scoring later"** |
| 11 | **Win shot:** back to **Today's Intel**, now with a FULLER feed of matched opps. Hold 3s. | Payoff = the thin feed from shot 1 is now full. |
| 12 | Point cursor at the **"Do it now →"** button (in edit, this is where you overlay the CTA). | "Your turn" close. |

## VIDEO 2 — Find your customers  (Market Research · 4–5 min)

| # | ON SCREEN — do this | Purpose / edit callout |
|---|---------------------|------------------------|
| 1 | **Hook shot:** open **Market Research** panel, empty search box, cursor in it. Hold 2s. | "Who buys it?" is the question. |
| 2 | **Type:** *"commercial HVAC installation"* → run. | Same plain-English search as Video 1 (continuity). |
| 3 | Let results load → show the **agencies ranked by dollars**. Scroll the top few. | Real spend, not a list of everyone. Callout: **"Ranked by real FY spend — USASpending"** |
| 4 | **Click into one agency** (pick one with strong HVAC spend). | Beat 2 — go a level deeper. |
| 5 | Show the **buying offices** under it. | "The VA buys somewhere" → "this office." |
| 6 | Open **Decision Makers / contacts** for that office. Scroll a couple real contacts. | The people to reach. Callout: **"Real contacts from award data"** |
| 7 | Click **Save to Target List** on the agency (and a contact if the UI allows). | Beat 3. |
| 8 | Navigate to **My Target List** (sidebar → Pipeline → My Target List). | Show it landed. |
| 9 | **Win shot:** the populated Target List — agencies + contacts in one place. Hold 3s. If any live badge (pain/opp/event) is showing, point at it. | Payoff = a real outreach list. Callout: **"Mindy keeps this live — new pains, buys, events"** |
| 10 | Point cursor at **"Do it now →"**. | CTA overlay in edit. |

## VIDEO 3 — Create your first bid  (My Pursuits · 5–6 min)

| # | ON SCREEN — do this | Purpose / edit callout |
|---|---------------------|------------------------|
| 1 | **Hook shot:** open **My Pursuits** with a few tracked opps. Hold 2s. | The pipeline of tracked work. |
| 2 | **Open one pursuit** (use one that has attachments/a solicitation, so bid→matrix→draft all work — NOT a bare Sources Sought with no docs). | ⚠️ Critical: pick a respondable notice with a body/attachments or the draft step dead-clicks. |
| 3 | Run **bid/no-bid** → show the fit score + strengths/gaps. | Beat 1. Callout: **"Scored against YOUR Vault"** |
| 4 | Point at where the **decision is recorded**. | Discipline most SMBs skip. |
| 5 | Open the **compliance matrix** → show extracted **shall/must** requirements as rows. Scroll a few. | Beat 2. Callout: **"Every requirement evaluators grade you on"** |
| 6 | Show a requirement **row you can assign/track** (owner, covered status). | Nothing slips. |
| 7 | Click **Draft** → show sections generating, mapped to requirements. | Beat 3. Callout: **"Drafted from YOUR real evidence"** |
| 8 | Run the **disqualifier scan** → show it flag/clear issues. | Catch it before submit, not after. |
| 9 | Click **Export .docx** → show the download, then **open the .docx** (Word/Pages). Scroll a page. | The artifact is real and openable. |
| 10 | **Win shot:** the finished .docx on screen. Hold 3s. | Payoff = a submission-ready response. |
| 11 | Point cursor at **"Do it now →"**. | CTA overlay in edit. |
