Mindy is an AI-powered federal market intelligence platform: daily opportunity ranking, agency forecasts, expiring contracts, teaming, a 317K-contractor database, pipeline tracking, and per-client workspaces. The problem this solves for GCAP

Your value is building capability, not handing out leads. And every quarter that progress has to be re-assembled by hand into a report for SBTDC / SBA.

Mindy makes the capability work the system of record, so the reporting is a by-product, not a second job.

What's included — and the two things that aren't

So there's no surprise at contracting, here is exactly where the flat license stops and the two optional add-ons begin:

✓ Included — unlimited

  • Every in-platform tool: Today's Intel, forecasts, expiring contracts, teaming, market research
  • The full 317K-contractor database — profiles, lookups, and capability searches
  • Per-client workspaces, pipeline, capability-progression tracking
  • Org admin, GCAP-branded tab, quarterly funder export
  • Onboarding, named contact, support, quarterly office hours

✗ Not in the flat license — optional add-ons

  • Proposal Assist — the AI proposal-drafting feature. Included with every counselor seat. For client workspaces it's an optional $99/mo per-user add-on the client buys directly — off by default, never billed to GCAP.
  • MCP / API credits — the programmatic credit system on our site (for connecting Mindy to outside AI tools) is a separate Pro-member product, sold as prepaid credit packs. The GCAP license is a seat license to work inside the platform; it does not include MCP/API credits. A counselor who wants the API channel buys credits like any Pro member.

Keeping both out of the flat license is exactly what lets the center price stay flat and budgetable — Proposal Assist is a simple $99/mo per-user add-on a client can turn on or off, and MCP/API credits are a separate Pro-member product. Neither ever changes what GCAP pays.
