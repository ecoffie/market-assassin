**Version:** 1.0
**Date:** May 4, 2026
**Author:** Claude Code
**Status:** Draft

## Executive Summary

Create a high-converting SaaS landing page for Market Intelligence (MI) at `govcongiants.com/mi`. This page will serve as the primary entry point for new users, replacing the current fragmented approach across multiple domains.

---

## Problem Statement

### Current Issues
1. No dedicated SaaS landing page on the marketing site (`govcongiants.org`)
2. `/market-intelligence` page on tools site is access/checkout focused, not conversion focused
3. No clear MI Free → MI Pro funnel with social proof and ROI positioning
4. Shop.govcongiants.org creates confusion (SaaS users don't "shop")
5. Multiple entry points with no consistent messaging

### Target State
Single, conversion-optimized landing page at `govcongiants.com/mi` that:
- Captures free tier signups (Daily Alerts + Opportunity Hunter)
- Converts to MI Pro ($149/mo)
- Positions against Deltek GovWin ($800-1,200/mo)
- Includes social proof, testimonials, and ROI calculator

---

## User Stories

### Primary Users
1. **Small GovCon firms** (1-10 employees) looking for affordable market intelligence
2. **Capture managers** tired of paying $800+/mo for enterprise tools
3. **New GovCon entrants** who need guidance on finding opportunities
4. **BD professionals** who want AI-powered daily briefings

### User Journeys

**Journey 1: Free Tier Signup**
```
Google search → Landing page → "Start Free" CTA → NAICS onboarding → Daily Alerts active
```

**Journey 2: Direct Pro Purchase**
```
Landing page → Tier comparison → "Get Pro" CTA → Stripe checkout → Dashboard access
```

**Journey 3: Free to Pro Upgrade**
```
Using Free tier → Sees Pro benefits in email → Clicks upgrade link → Checkout → Pro access
```
