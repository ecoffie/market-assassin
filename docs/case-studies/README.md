# Case Study Videos — Transcripts & Repurposing

Student case study video transcripts for content repurposing (emails, social posts, landing pages, etc.).

**→ [Video URLs (review list)](video-urls.md)** — All case study video links in one place for reviewers.

## Structure

Each case study is a markdown file with:
- **Metadata** — Student name, date, topic, tools used
- **Transcript** — Full video transcript
- **Key quotes** — Pull-quotes for repurposing
- **Repurposing notes** — Ideas for content spin-offs

## Naming convention

`{slug}.md` — e.g. `maria-first-contract-win.md`, `james-opp-hunter-breakthrough.md`
