### Section 3: How to Set Up SAM.gov Alerts (Pages 9-11)
**Step-by-step with screenshots**

1. **Login to SAM.gov** → beta.sam.gov → Sign In
2. **Navigate to Saved Searches** → Contract Opportunities → Create Saved Search
3. **Configure Your Filters:**
   - NAICS: 236220, 237110, 237310, 541330
   - Set-Aside: Small Business, 8(a), SDVOSB, HUBZone, WOSB
   - State: FL, GA, VA, TX (your target states)
   - Agency: DEPT OF DEFENSE, DEPT OF VETERANS AFFAIRS
4. **Set Alert Frequency:** Daily
5. **Name Your Search:** "AEC Construction Alerts"
6. **Recommended Saved Searches to Create:**
   - "USACE Construction SAT" (under $350K)
   - "NAVFAC MACCs" (keywords: MACC, MATOC)
   - "VA Healthcare Construction"
   - "Sources Sought - AEC" (notice type: Sources Sought)

**Pro Tips:**
- Create SEPARATE alerts for Sources Sought vs Solicitations
- Set up alerts for your COMPETITORS' names as keywords
- Include "Design-Build" and "DB" as keywords

### Section 4: How to Track Recompetes on USASpending.gov (Pages 12-14)
**Step-by-step with screenshots**

1. **Go to USASpending.gov** → Advanced Search → Contracts
2. **Filter by:**
   - NAICS Code: 236220
   - Awarding Agency: DEPT OF DEFENSE
   - Period of Performance Current End Date: 09/30/2026 - 03/30/2027
3. **Export to CSV** for your tracking spreadsheet
4. **Key Columns to Analyze:**
   - `recipient_name` = Incumbent
   - `current_total_value_of_award` = Contract size
   - `period_of_performance_current_end_date` = When to start engaging
   - `number_of_offers_received` = Competition level

**Rule of Thumb:**
- 12-18 months out: Start building relationships
- 9-12 months out: Sources Sought responses
- 6-9 months out: RFP drops
- 3-6 months out: Proposal writing
