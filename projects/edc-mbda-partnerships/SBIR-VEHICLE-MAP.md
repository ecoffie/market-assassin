## Anduril-path note (why SBIR, not CSO/OTA, for the KEY)
Only an SBIR/STTR award unlocks the **Phase III sole-source door** (15 U.S.C. §638(r)(4)) — uncapped,
no-recompete, every agency. CSO = one-off buy; OTA = one-off agreement; neither gives the key. So: do
ONE SBIR for the key (NV012 or a better-fit if DSIP surfaces one); use CSO/OTA/direct-buy for PARALLEL
revenue. You only need to WIN one SBIR.
