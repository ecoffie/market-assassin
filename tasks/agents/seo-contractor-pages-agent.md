# SEO Contractor Pages Agent

**Status:** Draft  
**Owner:** GovCon Giants / SEO + Product  
**Mission:** Identify, create, refresh, and monitor public contractor intelligence pages that attract qualified search traffic and convert users into MI.

## Job To Be Done

The agent answers:

- Which contractor pages should exist?
- Which pages are worth refreshing?
- Which data can be public?
- Which data should be gated?
- Are pages ranking, indexed, and converting?
- Which pages should link into MI workflows?

## Source Inputs

- Contractor database
- Award history
- Contractor sales chart data
- NAICS and agency summaries
- Recompete data
- Forecast data
- Search Console or SEO tool data
- Public competitor examples from HigherGov/GovTribe-style pages

## Candidate Page Signals

Prioritize contractors with:

- High total federal sales
- Recent awards
- Clear year-over-year sales trend
- Relevant NAICS search demand
- Upcoming recompetes
- Known teaming potential
- High customer research usage inside MI

## Public Page Output

Each public page should include:

- Contractor name
- Basic profile summary
- Year-over-year federal sales chart
- Top agencies
- Top NAICS
- Recent sample awards
- Data freshness note
- CTA into MI Free or MI Pro

## Gated MI Output

Full access should include:

- Complete sales history
- Download/export
- Contact details
- Recompetes
- Forecast connections
- Similar contractors
- Pipeline and teaming actions

## Cadence

### Weekly

- Identify new page candidates
- Refresh top pages
- Detect broken pages or stale data
- Recommend internal links

### Monthly

- Rank traffic growth
- Rank conversion performance
- Identify content gaps
- Recommend new clusters by NAICS, agency, or contractor type

## Guardrails

- Public pages must not expose private customer data.
- Public pages should use `govcongiants.com` unless they are authenticated app surfaces.
- App routes and authenticated workflows should use `mi.govcongiants.com`.
- Every public data point needs a source/freshness note.
- Avoid promising contract wins; frame MI as helping users find, evaluate, and pursue better opportunities.

## Definition Of Done

The agent is useful when it can produce:

- A ranked list of contractor pages to build
- A public/gated data plan for each page
- Internal linking suggestions
- SEO and conversion notes
- Product follow-ups for MI contractor research
