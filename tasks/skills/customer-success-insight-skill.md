# Customer Success Insight Skill

**Status:** Draft  
**Owner:** GovCon Giants / Customer Success  
**Purpose:** Turn MI user behavior into specific rescue, activation, upgrade, or white-glove actions.

## When To Use This

Use this skill when reviewing users, cohorts, activity logs, email engagement, profile completion, or paid account health.

Trigger examples:
- "Who should Annelle and Sikander contact?"
- "Which users are stuck?"
- "Which Pro users are not getting value?"
- "Who is ready for white-glove?"
- "How do we increase engagement?"

## Core Outcome

The customer success team should know who needs help and what action to take without waiting for Eric to interpret the data.

## User Signals

| Signal | Meaning | Action |
| --- | --- | --- |
| Entitled, no auth account | User cannot access MI yet | Send account setup invite |
| Account, no profile | MI cannot personalize value | Send profile setup nudge |
| Profile complete, no login | Setup worked, habit did not start | Send "first value" walkthrough |
| Email open, no click | Interest without action | Improve CTA or send direct deep link |
| Click, no app time | Product friction | Ask what blocked them |
| High app time, no saves | Browsing without decision | Teach save/track workflow |
| Saved opportunities | Moving toward outcome | Offer proposal/team support |
| Pro inactive | Retention risk | Customer success rescue call |
| High usage + paid history | Strong value signal | Founder call or white-glove invite |

## Segments

- MI Free activation
- MI Pro onboarding
- Bundle customer activation
- Internal user testing
- White-glove candidates
- 10-10 Forever candidates
- At-risk paid users
- Audience-only users

## Outreach Rules

1. Do not treat all users the same.
2. Tie every message to the last meaningful behavior.
3. Lead with helping them get a result, not explaining the product.
4. Do not pressure inactive free users before they show intent.
5. Prioritize paid users, high-intent users, and users moving toward proposals.

## Recommended Actions

| Segment | Message Angle | Owner |
| --- | --- | --- |
| No profile | "Let us help MI find better matches for you." | Customer Success |
| Opened alerts | "Which of these opportunities was worth a closer look?" | Customer Success |
| Viewed contractors | "Are you researching competitors or partners?" | Coach/Sales |
| Saved pipeline item | "Do you want help deciding bid/no-bid?" | Coach |
| High-value Pro user | "Let's map your next 90 days." | Eric/Coach |
| Enterprise buyer | "Let's discuss team rollout and package fit." | Branden |

## Output Format

When using this skill, produce:

- Cohort summary
- Top users to contact
- Why each user qualifies
- Recommended message
- Owner
- Priority
- Follow-up tag
- Data quality caveats

## Definition Of Done

The insight is useful when a team member can take action today:

- Send a setup nudge
- Book a support call
- Invite to MI Pro
- Route to a coach
- Route to Branden for enterprise/package sales
- Route to Eric for 10-10 Forever relationship building
